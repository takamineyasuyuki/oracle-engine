# 🌟 日本語UIの使い方

## 超簡単！ブラウザで使う方法

サーバーを起動したら、以下の手順で日本語UIを開けます：

### 方法1: ダブルクリックで開く（一番簡単！）

1. Finderで `oracle_engine` フォルダを開く
2. **`index.html`** ファイルを探す
3. `index.html` を**ダブルクリック**
4. Safariで開きます！

### 方法2: ブラウザから直接開く

1. Safariを開く
2. アドレスバーに以下を入力：
   ```
   http://localhost:8000/
   ```
3. ページが開いたら、アドレスバーに以下を入力：
   ```
   file:///Users/あなたのユーザー名/Downloads/oracle_engine/index.html
   ```

### 方法3: サーバーから配信（推奨）

サーバーが起動している状態で：

1. Safariを開く
2. アドレスバーに以下を入力：
   ```
   http://localhost:8000/
   ```
3. 表示されたページから `index.html` のリンクをクリック

## 🎯 使い方の流れ

### STEP 1: MBTIを選択
- 16個のボタンから自分のMBTIタイプを選ぶ
- わからない場合は [16personalities.com](https://www.16personalities.com/ja) で診断

### STEP 2: デバイスを選択
- 🍎 **Mac**: MacBook / iMac をお持ちの方
- 🪟 **Windows PC**: Windows PCをお持ちの方
- 📱 **スマホのみ**: PCを持っていない方

### STEP 3: 5問の質問に答える
- 1〜5のスケールで答えます
  - **1-2**: 左側の特性（例: 慎重に従う）
  - **3**: 中間
  - **4-5**: 右側の特性（例: 積極的に破壊する）

### 診断結果
以下が表示されます：
- 🎯 **あなたのアーキタイプ**（8タイプから診断）
- ✅ **ビジネス上の強み**
- ⚠️ **ビジネス上の弱み**
- 📰 **2026年2月の最新AI副業トレンド**（リアルタイム検索）
- 🗺️ **あなた専用の3ヶ月ロードマップ**
- 🛠️ **開発中の自動化ツール情報**

## ⚠️ 重要: サーバーを起動しておく

index.htmlを開く**前に**、必ずサーバーを起動してください：

```bash
cd oracle_engine
source venv/bin/activate
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

サーバーが起動していないと、診断ボタンを押してもエラーになります。

## トラブルシューティング

### 「診断結果を見る」を押してもエラーになる
→ サーバーが起動していない可能性があります
```bash
# ターミナルでサーバーを起動
uvicorn app.main:app --reload
```

### ページが真っ白
→ サーバーのURLが間違っている可能性があります
→ `index.html` をダブルクリックで開いてください

### CORSエラーが出る
→ サーバーを起動してから、もう一度試してください
